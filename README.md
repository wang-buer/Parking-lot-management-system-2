# 停车场管理系统2

## 1、项目介绍

停车场管理系统拥有两种角色

管理员：管理员管理、用户管理、车位管理、停车费设置、停车缴费、用户充值、停车记录查询

用户：登录注册、添加车辆、停车、查看停车费和卡余额、查看停车历史记录


## 2、项目技术

后端框架： Servlet、mvc模式

前端技术：jsp、css、JavaScript、JQuery

## 3、开发环境

- JAVA版本：JDK1.8
- IDE类型：IDEA、Eclipse都可运行
- tomcat版本：Tomcat 7-10版本均可
- 数据库类型：MySql（5.x和8.x版本都可） 
- 硬件环境：Windows 或者 Mac OS
